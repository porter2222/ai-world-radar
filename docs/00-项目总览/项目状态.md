## 5. 核心产品结论

AI World Radar 的一句话定位是：

> AI World Radar 是一个面向中文用户的全球 AI 圈事件雷达，它从外网和中文公开信息源中发现 AI 圈热点与重要事件，并将碎片化信息整理成中文用户能快速理解、可追踪、可修正的事件档案。

P1 不以新闻链接、文章摘要或日报为核心，而是把全球 AI 圈的碎片化信号整理成可理解、可追踪、可修正的事件档案。

晨报不取消，但后置到 P1.5，基于已发布事件生成。

## 6. 最新 Agent 架构结论

新版 Agent 系统设计 v1.0 的核心口径是：

> AI World Radar 是基于 LangGraph 的编辑部式 Multi-Agent 情报生产系统。

终局架构：

```text
LangGraph 工作流编排
+ 编辑部式主 Agent
+ 能力型子 Agent / Skill
+ 工程 Tool 白名单
+ PostgreSQL 状态与审计记录
+ 后台监控与人工修正
```

P1 实现：

```text
单机 Python Worker
+ PostgreSQL
+ 本地文件缓存
+ 同步 pipeline
+ 最小 Agent 闭环
+ 最小运行记录
```

三类主 Agent：

- 值班编辑 Agent：负责信号筛选、事件归并、排序和选题角度。
- 研究写作 Agent：负责资料整理、首页事件卡、详情正文和修订。
- 审稿发布 Agent：负责质量门禁、风险检查、修订意见和发布建议。

核心边界：

- Agent 负责语义判断和内容生产。
- 工程系统负责采集、入库、状态流转、发布、审计和兜底。
- Agent 不能直接发布，不能直接修改核心数据库状态。
- P1 不做完整真实性核验，不做复杂社媒大爬虫，不做追问助手。
