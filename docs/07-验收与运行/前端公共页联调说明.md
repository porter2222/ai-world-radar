# 前端公共页联调说明

更新时间：2026-06-23

## 范围

本说明覆盖 AI World Radar 第一版公共前端：

- 首页事件流：`apps/web/app/page.tsx`
- 事件详情页：`apps/web/app/events/[slug]/page.tsx`
- FastAPI 产品接口适配：`apps/web/lib/product-api.ts`

当前不包含：

- Agent 运行 / 审稿后台管理页
- 登录、权限、支付、部署
- 前端直连 PostgreSQL
- 后端 worker 代码修改

## 启动后端 Product API

在 `apps/worker` 目录启动 FastAPI：

```powershell
uvicorn worker.api.app:create_app --factory --host 127.0.0.1 --port 8016
```

默认前端会读取：

```text
http://127.0.0.1:8016
```

如需修改后端地址，在 `apps/web` 启动前设置：

```powershell
$env:AI_WORLD_RADAR_API_BASE_URL="http://127.0.0.1:8016"
```

## 启动前端

在 `apps/web` 目录运行：

```powershell
npm install
npm run dev
```

默认访问：

```text
http://127.0.0.1:3000
```

## 接口字段映射

### 首页事件卡

接口：`GET /events?limit=20&offset=0`

| 后端字段 | 前端用途 |
| --- | --- |
| `slug` | 详情页链接 `/events/{slug}` |
| `title` | 事件标题 |
| `card_summary` | 首页卡片摘要 |
| `category` | 分类标签；为空时显示 `AI 事件` |
| `signal_label` | 轻量信号标签 |
| `cover_image_url` | 封面图；为空时显示分类占位 |
| `source_hint` | 右侧来源提示，直接使用后端文案 |
| `source_count` | 暂不直接展示，作为后续备用 |
| `published_at` | 前端转换为相对时间 |

首页不读取完整 `source_refs`，避免列表页 N+1 详情请求。

来源提示由前端强制单行展示：

```css
white-space: nowrap;
overflow: hidden;
text-overflow: ellipsis;
text-align: right;
```

### 事件详情页

接口：`GET /events/{slug}`

| 后端字段 | 前端用途 |
| --- | --- |
| `title` | 详情页标题 |
| `detail_summary` | 详情摘要 |
| `detail_body` | 正文；按空行切成自然段 |
| `why_it_matters` | “为什么值得看”说明 |
| `follow_up_points` | “继续观察”列表 |
| `source_refs` | 完整来源链接列表 |
| `category` | 分类标签；为空时显示 `AI 事件` |
| `signal_label` | 轻量信号标签 |
| `cover_image_url` | 封面图；为空时显示分类占位 |
| `published_at` | 前端转换为相对时间 |

## 当前实现说明

- 首页和详情页是 Next.js App Router server component。
- `apps/web/lib/product-api.ts` 只通过 FastAPI Product API 读取数据。
- `cover_image_url` 为空时，前端使用克制的分类占位，不触发图片生成。
- `source_hint` 直接按后端返回展示，不在前端二次改写文案。
- 详情页的 `source_refs` 是完整来源轨迹，保留给用户验证。
