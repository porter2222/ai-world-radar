# Public Web Events Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first Next.js public web frontend for AI World Radar, covering the homepage event feed and independent event detail page backed by the current FastAPI Product API.

**Architecture:** Add `apps/web` as a small Next.js App Router application. The web app reads `GET /events` and `GET /events/{slug}` through a typed API adapter and renders product-facing pages only; it does not call PostgreSQL directly and does not expose Agent/admin concepts.

**Tech Stack:** Next.js, React, TypeScript, Vitest, Testing Library, FastAPI Product API.

---

## Scope

Build only:

- Public homepage event feed.
- Public event detail page.
- API client and formatting helpers for the current backend contract.
- Documentation for running and integrating the web app.

Do not build:

- Agent management page.
- Admin review actions.
- Login, permission, payment, deployment.
- Backend or worker code changes.
- Mobile-specific redesign.

## File Structure

- Modify `.gitignore`: ignore web generated artifacts such as `node_modules`, `.next`, `coverage`, and npm debug logs.
- Create `apps/web/package.json`: frontend scripts and dependencies.
- Create `apps/web/package-lock.json`: npm lockfile generated by install.
- Create `apps/web/next.config.mjs`: minimal Next.js config.
- Create `apps/web/tsconfig.json`: TypeScript config.
- Create `apps/web/vitest.config.ts`: jsdom-based test config.
- Create `apps/web/app/layout.tsx`: root HTML shell.
- Create `apps/web/app/page.tsx`: homepage server component.
- Create `apps/web/app/events/[slug]/page.tsx`: event detail server component.
- Create `apps/web/app/globals.css`: product UI tokens, layout, components, single-line source hint rules.
- Create `apps/web/components/EventCard.tsx`: homepage card component.
- Create `apps/web/components/EventDetail.tsx`: detail page component.
- Create `apps/web/lib/product-api.ts`: typed FastAPI client.
- Create `apps/web/lib/format.ts`: relative time, source formatting, paragraph splitting helpers.
- Create `apps/web/lib/*.test.ts` and `apps/web/components/*.test.tsx`: behavior tests.
- Create `docs/07-验收与运行/前端公共页联调说明.md`: runbook and field mapping.
