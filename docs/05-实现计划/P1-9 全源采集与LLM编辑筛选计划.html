筛选策略

  1. 全源采集：全部 enabled sources 写入 source_signals。
  2. 硬过滤：去空标题、无追踪 URL、重复 hash / URL、过旧和已处理信号。
  3. 候选分组：按 canonical URL、repo / product / model 名和标题近似合并。
  4. LLM Selector：从 candidate groups 中选 Top N，输出 selected / rejected / reason。
  5. 后续 pipeline：只对 selected group 运行 writer / reviewer。
